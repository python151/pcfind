How does the dark web work?
TOR (the onion router), better known as the dark web is a network of computers that act as nodes which allow anonymous encrypted communications. TOR works by first encrypting the data through layers. Then sending it through the nodes peeling off one layer of encryption each time it goes through a node only to find where it will go next finally exiting the TOR network going to your website. It will then do the reverse back to you.
TOR is used to maintain anonymousity for informants, and whistleblowers, and allows people to avoid persecution for doing things online if it is restricted in there government.
